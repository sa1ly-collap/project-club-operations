## Информационная система учёта и анализа деятельности компьютерного клуба

### 1. Контекст
Учебный проект по проектированию системы для операционного учёта работы компьютерного клуба. Сейчас сотрудники используют разрозненные инструменты (Telegram, Google‑таблицы, LanGame, FusionPos); новая система должна объединить данные, автоматически формировать отчёты по сменам, уборкам и выручке и напоминать о плановых задачах.

### 2. Цель и задачи
Цель. Спроектировать информационную систему, которая объединяет разрознённые источники данных и автоматизирует операционный учёт работы клуба.
Задачи:
– описать текущие процессы учёта и отчётности (as‑is) и выявить проблемы ручного ведения данных;
– спроектировать целевые процессы (to‑be) с использованием единой системы;
– определить ключевые функции ИС: автоматические отчёты по сменам и уборкам, фиксация времени и фотоотчётов, напоминания о плановых задачах;
– наметить план внедрения системы по этапам с оценкой сроков.

### 3. Модели и диаграммы
В проекте использованы следующие артефакты:
- BPMN‑диаграмма текущих процессов учёта и отчётности «as‑is».
- BPMN‑диаграмма целевых процессов «to‑be» с использованием информационной системы.
- UML use case‑диаграмма для описания сценариев работы пользователей с системой.
- Диаграмма Ганта в MS Project с этапами анализа, проектирования и внедрения.

### 4. Результаты
Проанализирована текущая схема операционного учёта, где значительная часть данных по сменам, уборкам и выручке ведётся вручную в разных инструментах, что повышает риск ошибок и потери информации.  
Целевая модель процессов и проект единой информационной системы показывают, как можно сократить объём ручных операций, повысить прозрачность учёта, ускорить подготовку отчётов и упростить доступ к данным для управляющего клубом.

### 5. Используемые инструменты
- BPMN — описание процессов учёта и отчётности as‑is и to‑be.
- UML — описание пользовательских сценариев и функциональных требований к системе.
- MS Project — диаграмма Ганта с этапами проекта по внедрению ИС.
- Miro / другие средства визуализации — работа с диаграммами и согласование схем.
